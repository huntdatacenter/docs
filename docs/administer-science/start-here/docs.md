# About docs

**These web pages serves as our user documentation (docs).**

[[toc]]

We have split our user documentation in three distinct sections aimed at different users groups inside our community: do science, administer science and govern science. 

### Do science

This section is aimed a scientists (lab users) that are actively doing analysis in a lab. The information is aimed at practical "how to" guides for workbench, technical tools, analytical tools, machine learning, data transfers etc. This section does also include guides for technical troubleshooting and frequently asked questions.

### Administer science

This section is aimed at scientific administrators that are managing research projects that uses HUNT Cloud. The information is aimed to be non-technical with a focus on services, prices, data and agreements. 

### Govern science 

This section is aimed at security and privacy officers at organizations that have scientists that uses HUNT Cloud. The information is focused on security and privacy policies, ISO certification and risk management.
