# Lab mangement

# New lab

Click the button below to order a new lab under your existing data space.

Request a new lab

  • Who can order: Data space leaders and data space compliance coordinators in active data spaces.
  • Required attachment: A signed lab order, lab users in separate orders (see above).
  • Expected response time: One week.
  • Cost: One lab subscription as specified in the Services specifications and Price list.

# Transport

Internal transfers and import

Data transfers inside HUNT Cloud and external data imports can be ordered from our lab orders page.

# External export kista

The external export kista allows labs to make data available for external parties outside HUNT Cloud in a compliant manner.

Request an external export kista

  • Who can order: Data space leaders or Data space compliance coordinators.
  • Required attachments: (1) A signed external kista export order, and (2) one SSH public key from the external uploader.
  • Expected response time: Days.
  • Expected delivery: Access information sent to the lab user that will make data available for export.
  • Cost: Included in your lab subscription.

# Network opening

Network opening orders allows data transfers directly to and from parties outside HUNT Cloud.

Request a network opening

  • Who can order: Data space leaders or Data space compliance coordinators.
  • Required information: IPv4 address and port number for the external party.
  • Required attachments: A signed network opening order.
  • Expected response time: Days.
  • Expected delivery: Network opening communicated in your Slack lab channel.
  • Cost: Included in your lab subscription.
