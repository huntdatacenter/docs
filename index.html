# Welcome to the HUNT Cloud Documentation

HUNT Cloud is a scientific computing service located at NTNU (opens new window) in Norway.

These pages contains official user documentation on our cloud services. The content is aimed at lab users.

Read our eBook (opens new window) and visit our HUNT Cloud (opens new window) pages for general information on our services. See our latest Ting report for a detailed status update.
