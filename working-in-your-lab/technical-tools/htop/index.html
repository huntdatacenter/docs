# HTOP

Htop is an interactive processes viewer for Linux systems. It is practically a top-like tool, but it displays information about system in colourful and intuitive way, and allows for output scrolling. You can read up more in the comparison article Htop versus Top in Linux or follow straight technical explanations in Htop explained - Explanation of everything you can see in htop/top on Linux.

# Linux

sudo apt-get update -y && sudo apt-get install -y htop

# MacOS

REQUIREMENT

Install Homebrew package manager for macOS: brew

brew install htop
