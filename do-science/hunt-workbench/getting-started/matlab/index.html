# MATLAB

You can run MATLAB both from Jupyterlab or from the remote desktop (VNC) in your Workbench.

To be able to run MATLAB in Workbench you need to bring your own license. In most cases such license is already provided by your institution and you can request it from your IT department.

NTNU employees

We already have the license for NTNU employees and it should already be in your workbench, if you can't find it, send us a message through general service request in do science Service desk (opens new window).

# In JupyterLab

MATLAB installation in Workbench comes with the MATLAB kernel, you can read about it on this page (opens new window).

# In Remote desktop

MATLAB is available in your Workbench VNC desktop under the "Education" tab.
